# Installation

## From PyPI

```bash
pip install mlprep
```

## From Source

```bash
git clone https://github.com/takurot/mlprep.git
cd mlprep
pip install .
```

## Requirements

- Python 3.10+
- Rust toolchain (for building from source)

## Verify Installation

```bash
mlprep --help
```

```bash
python -c "import mlprep; print(mlprep.__version__)"
```
